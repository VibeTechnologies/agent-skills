# VibeBrowser Agent Skills

Reusable capabilities for AI agents working with [VibeBrowser](https://vibebrowser.app).

## Installation

```bash
npx skills add VibeTechnologies/agent-skills
```

## Available Skills

### vibebrowser-cli

Control a browser connected through the Vibe relay. Use for any request that says go to/open/visit a URL or domain, or asks to read/extract information from a webpage in the relay-backed local browser session.
